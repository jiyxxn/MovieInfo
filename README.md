## 💬 프로젝트 소개

: TMDB Open API를 활용한 영화 소개 웹 사이트입니다.

<br>
<br>

## ⚙ 프로젝트 기능 소개

- jQuery 라이브러리 사용 없이 🍦 바닐라 자바스크립트로 구성된 프로젝트입니다.
- TMDB Open API를 이용하여 인기 영화 데이터를 활용했습니다.
- 실시간 데이터를 이용해 카드 UI를 구성했습니다.
- 세션 스토리지를 이용, 북마크 기능을 구현해 유저가 북마크한 영화를 모아볼 수 있습니다.
